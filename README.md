# Will you be my Valentine?

A small Valentine's proposal page (HTML + CSS + JS only).

## Change the name

- **In code:** Edit `LVR_NAME` at the top of `script.js` (default: `'Neha'`).
- **Via URL:** Open the page with `?name=YourName`, e.g. `index.html?name=Neha`.

## Run locally

Open `index.html` in a browser, or:

```bash
npx serve .
```

Then open the URL shown (e.g. http://localhost:3000).

## Deploy

Upload the folder to GitHub Pages, Netlify, or any static host. For a custom name via env on Netlify/Vercel, replace `LVR_NAME` at build time if needed.
